# Code of Conduct

Be direct, respectful, and practical.

This project is a Windows system tool, so reviews may be strict about safety,
correctness, and clarity. That strictness should stay focused on the work, not
on people.

Unacceptable behavior includes harassment, threats, doxxing, spam, and hostile
or discriminatory comments.

Maintainers may remove comments, close issues, or block contributors who make
the project unsafe or unpleasant to work on.
